__Set/Read realtime clock__

Set time:

	RT p1p2p3p4p5p6

Get time:

	RT

Returns: RT p1p2p3p4p5p6

| p   | Function |
| --- | --- |
| 1 | YY year, 00-99 for 2000-2099                  |
| 2 | MM month, 01-12 for Jan-Dec                   |
| 3 | DD day, 01-31                                 |
| 4 | HH hour, 00-23                                |
| 5 | MM minute, 00-59                              |
| 6 | SS second, 00-59 (leap seconds not supported) |
