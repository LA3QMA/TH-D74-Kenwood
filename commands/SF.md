__Set/Read step size__

Set step size:

	SF p1,p2

Get step size:

	SF p1

Returns: SF p1,p2

| p1  | [Band](/tables/band.md) |
| --- | --- |
| 0 | Band A |
| 1 | Band B |

| 8 | 25 kHz   |

9 kHz is limited to range 0.100 - 1.709
