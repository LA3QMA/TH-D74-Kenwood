__Set/Read TNC mode__

Set TNC mode:

	TN p1,p2

Get TNC mode:

	TN

Returns: TN p1,p2

| p1  | Function |
| --- | --- |
| 0 | Off  |
| 1 | APRS |
| 2 | KISS |

| p2  | [Band](/tables/band.md) |
| --- | --- |
| 0 | Band A |
| 1 | Band B |

Key: (F) APRS

__Not possible to get out of KISS mode__
*Maybe sending the KISS exit command: C0 FF C0 (have to try this)*
