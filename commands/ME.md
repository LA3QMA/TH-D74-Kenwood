__Set/Read the memory channel__

Set the memory channel:

	ME p1, p2, p3, p4, p5, p6, p7, p8, p9, p10, p11, p12, p13, p14, p15, p16, p17, p18, p19, p20, p21, p22

Read the memory channel:

	ME xxx
	
Returns: memory channel number (3 digit)
