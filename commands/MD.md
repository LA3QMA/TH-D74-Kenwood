__Set/Read mode__

Set mode:

	MD p1,p2

Get mode:

	MD

Returns: MD p1,p2

| p1  | [Band](/tables/band.md) |
| --- | --- |
| 0 | Band A |
| 1 | Band B |

| p2  | [Mode](/tables/mode.md) |
| --- | --- |
| 0 | FM   |
| 1 | DV   |
| 2 | AM   |
| 3 | LSB  |
| 4 | USB  |
| 5 | CW   |
| 6 | NFM  |
| 7 | DR   |
| 8 | WFM  |
| 9 | R-CW |

Key: MODE
