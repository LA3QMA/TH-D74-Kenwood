__Set/Read memory/VFO mode__

Set memory/VFO mode:

	VM p1,p2

Get memory/VFO mode:

	VM p1

Returns: VM p1,p2

| p1  | [Band](/tables/band.md) |
| --- | --- |
| 0 | Band A |
| 1 | Band B |

| p2  | Function |
| --- | --- |
| 0 | VFO mode    |
| 1 | Memory mode |
| 2 | Call mode   |
| 3 | DV mode     |

Keys: VFO, MR, CALL, MODE
