__Set/Read internal GPS mode__

Set GPS mode:

	GP p1,p2

Get GPS mode:

	GP

Returns: GP p1,p2

| p1  | GPS [status](/tables/status.md) |
| --- | --- |
| 0 | Off |
| 1 | On  |

| p2  | Data output ("PC Output") [status](/tables/status.md) |
| --- | --- |
| 0 | Off |
| 1 | On  |
